# 10875

- 피보나치 수열 : 재귀 2개의 합을 구하기 때문에, 초기 `n == 0`, `n == 1` 두개의 값의 리턴 조건이 필요

```
int fibonacci(int number)
{
    if (number == 0)
        return 0;
    if (number == 1)
        return 1;
    else
        return fibonacci(number - 1) + fibonacci(number - 2);
}
```
